# Jogo Snake.

Jogo desenvolvido puramente com HTML, CSS e Javascript com o objetivo de treinar a manipulção da DOM.

A aplicação possui:
- Contador de pontuação 
- Incremento de dificuldade com o passar dos níveis.
- Modal com resumo da partida
